About the Project

The Virtual Mouse is an innovative web-based application that allows users to control mouse functions using hand gestures and a webcam. Developed using Python, OpenCV, MediaPipe, and Flask, the project uses real-time hand tracking to interpret finger positions as mouse movements and clicks.


This project is designed to provide an alternative method of interaction, enhancing accessibility and showcasing the power of computer vision. Whether it's for hands-free navigation or as a proof of concept for gesture-controlled interfaces, the Virtual Mouse demonstrates how modern technologies can reshape user interaction.
